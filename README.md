# n8n Tech News Automation

This project is an automated workflow built using n8n.

## What it does

* Fetches latest technology news from NewsAPI
* Selects top 3 headlines
* Formats them into a single message
* Sends the message to Telegram daily

## Tech Stack

* n8n
* NewsAPI
* Telegram Bot API

## Automation

Runs daily at a scheduled time using Cron (Schedule Trigger).

## Workflow

The workflow is included as a JSON file in this repository.
